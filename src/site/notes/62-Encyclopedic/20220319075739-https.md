# HTTPS

> [!word] HTTPS
> The **HTTPS** protocol is a set of rules that permists two or more parties to communicate.
>
> - The lock and the protocol indicate that data is encrypted.
> - Browsers know to trust HTTPS website based on digital certificates, which are electronic documents that prove the identity of a website.
