# Manjaro File Hierarchy

- `/etc/`
  - `lightdm/`
	- `lightdm.conf`
	  - Greeter setup [Install Display Managers - Manjaro](https://wiki.manjaro.org/index.php/Install_Display_Managers)
	  - Login Customisation
  - `pacman.conf`
	- Setup `pacman`
	- uncomment `Color` for `yay` and `pacman` color
