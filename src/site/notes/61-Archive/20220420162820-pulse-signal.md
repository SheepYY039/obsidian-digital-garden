# Pulse Signal

> [!word] Pulse Signal
> Formed by a rapid or sudden transient changes between its high and low levels.
>
> - Looks like a rectangular waveform

![Pasted image 20220420162921](Pasted-image-20220420162921.png)

| Term         | Expression    |
| ------------ | ------------- |
| High Time    | $H$           |
| Low Time     | $L$           |
| Time Period  | $T=H+L$       |
| Frequency    | $\frac{1}{T}$ |
| Pulse Height | $V_{H}-V_{L}$ |
| Duty Cycle   | $\frac{H}{T}$ |
