# Duty Cycle

> [!word] Duty Cycle
> The **percentage** that describes the amount of **High Time ($H$)** over one **Time Period** > $$\text{Duty Cycle (\%)} = \frac{\text{Pulse Width}}{\text{Time Period}} = \frac{H}{T}$$
