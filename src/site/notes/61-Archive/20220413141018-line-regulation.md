# Line Regulation

> [!word] Line Regulation
> Line regulation measures **how stable the output voltage is** with respect to the **change in input voltage**.
>
> - Ideally, It should equal to 0

$$\text{Line regulation}=\frac{\Delta V_{O}}{\Delta V_{I}}$$
