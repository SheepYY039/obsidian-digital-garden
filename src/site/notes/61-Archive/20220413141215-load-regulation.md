# Load Regulation

> [!word] Load Regulation
> Load regulation meansures how **stable the output voltage is** with respect to the **change of output current**.
>
> - The ideal case: = 0
>   $$\text{Load regulation}=\frac{\Delta V_{O}}{\Delta I_{O}}$$
