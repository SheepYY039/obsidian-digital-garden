# Manjaro Theming

## Qt Apps

Make Qt Apps use GTK+ font & theme settings using **qt5ct**

```bash
pamac install qt5ct
```

```bash
# ~/.profile
export QT_QPA_PLATFORMTHEME="qt5ct"
```
